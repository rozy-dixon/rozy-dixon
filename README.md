# Rosalind Dixon

Rosalind Dixon is an artist, professional game designer, and first-year PhD student at the University of California, Santa Cruz, where she is pursuing a degree in Computational Media. She is interested in digital media, platform studies, and erotics, each discipline working to construct and develop an interdisciplinary, embodied feminist history.
